# Lista de Exercícios - C#

Exercícios desenvolvidos em C# para a disciplina de Programação Orientada a Objetos.

## Conteúdo abordado

- Tipos primitivos
- Estruturas condicionais
- Métodos
- Parse e TryParse
- Operações matemáticas

## Exercícios

1. Soma de dois números
2. Verificação de maioridade
3. Número par ou ímpar
4. Calculadora simples
5. Cálculo de média
6. Método de saudação
7. Cálculo do quadrado de um número
8. Uso de TryParse
9. Conversão de temperatura
10. Verificação de número positivo ou negativo

## Linguagem utilizada

C#
